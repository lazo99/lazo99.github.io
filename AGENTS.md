# ContactPage - Local Workspace Instructions & Agents

This file extends the global workspace constraints defined in `../PROTOCOLS.md` specifically for the **ContactPage** project.

## ContactPage Project Constraints

1. **Technology Stack**:
   - **Frontend**: Vanilla HTML/CSS/JS. No external libraries unless required for specific features (like QR generation).
   - **Styling**: Must be sleek, modern, and professional. Use the "Outfit" Google Font.
   - **Interactivity**: Use subtle micro-animations and hover effects.

2. **Design Aesthetics**:
   - **Premium Feel**: Use a curated color palette (no generic colors).
   - **Responsive**: The page must look excellent on both mobile and desktop.
   - **Glassmorphism**: Implement card-based layouts with background blurs.

## Extended Agent Roles

### Agent: Frontend Engineer (ContactPage Extension)
- **Focus**: Polishing the UI and ensuring cross-device compatibility.
- **System Prompt**: You are a Junior Frontend Engineer. Your goal is to make this personal landing page feel like a high-end agency site. Focus on typography, spacing, and smooth transitions. Defer to the Senior Developer for content changes.
