# Personal Landing Page

A professional contact/landing page for Andrew Lazo - Technical Support Engineer, Event Coordinator, Digital Media Specialist.

## Features

- Modern glassmorphism design with animated background effects
- Professional profile with 15+ years experience summary
- Three expertise sections: Technical, Project/Event Management, Creative/Digital Media
- Contact information with QR code for easy vCard download
- Fully responsive with accessibility enhancements
- Open Graph meta tags for social media sharing

## Live Demo
[https://lazo99.github.io](https://lazo99.github.io)

[https://github.com/lazo99/lazo99.github.io](https://github.com/lazo99/lazo99.github.io)

## Tech Stack
- HTML5 (semantic, ARIA labels, Open Graph)
- CSS3 (glassmorphism, animations, responsive)
- JavaScript (vCard generation, QR code)

## Local Setup
Clone the repository and open `index.html` in your browser:
```bash
git clone https://github.com/lazo99/lazo99.github.io.git
cd lazo99.github.io
open index.html
```
